# Seuraava Bussi

A real-time public transit departure board for Finland. Shows upcoming buses, trams, and trains from stops near your location.

**Live at: [https://www.seuraavabussi.fi/](https://www.seuraavabussi.fi/)**

## Supported Regions

- Helsinki (HSL)
- Tampere (Nysse)
- Turku (Foli)
- Jyvaskyla (Linkki)
- Oulu

## Features

- Real-time GPS-based departure times
- Automatic location detection
- Search any address or place
- Auto-refresh every 30 seconds
- Free to use

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- Digitransit and Waltti APIs
